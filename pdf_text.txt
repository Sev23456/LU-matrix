Задание: Сравнение методов решения СЛАУ
Метод Гаусса и LU-разложение
1 Цель работы
Реализовать набор функций для решения систем линейных алгебраи-
ческих уравнений (СЛАУ) методом Гаусса и методом LU-разложения.
Провести сравнительный анализ времени работы алгоритмов и проде-
монстрировать преимущество LU-разложения при решении систем с оди-
наковой матрицей, но разными правыми частями.
2 Задачи
Необходимо реализовать следующие алгоритмы:
•Решение СЛАУ методом Гауссабез выбора ведущего элемента.
•Решение СЛАУ методом Гауссас выбором главного элемента
по столбцу(частичный выбор).
•LU-разложениематрицы.
•Решение треугольных систем (прямая и обратная подстановки) для
использования совместно с LU-разложением.
Примечание: выбор ведущего элемента реализуется только в вариантах
метода Гаусса; LU-разложение выполняется без перестановок.
3 Требования к реализации
•Язык программирования — любой. Для получения дополнитель-
ных баллов рекомендуется реализовать программу на C или C++.
•Код должен быть структурирован: отдельные функции для каж-
дого метода, для замера времени, для генерации тестовых матриц,
для вычисления невязки и т.д.

4 План экспериментов
4.1 Сравнение времени решения одной системы
Для размеровn= 100,200,500,1000(или до максимально возможного с
учётом производительности вашего компьютера):
1. Сгенерировать случайную матрицуA(например, элементы из рав-
номерного распределения на[−1,1]) и случайную правую частьb.
2. Измерить время решения методом Гаусса без выбора ведущего эле-
мента.
3. Измерить время решения методом Гаусса с выбором ведущего эле-
мента.
4. Измерить суммарное время LU-разложения + решения (прямая и
обратная подстановки).
5. Для LU-разложения замерить отдельно время разложения и время
решения (подстановки).
Результаты оформить в виде таблицы (можно в формате Markdown или
CSV).
